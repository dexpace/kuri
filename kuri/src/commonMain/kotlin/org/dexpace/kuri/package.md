# Package org.dexpace.kuri

Root package for the **kuri** URI/URL parsing and manipulation library.

The sole public declaration in this package is currently [Kuri], a metadata placeholder
that carries the library version constant and confirms the build, source-set hierarchy,
and quality gates are wired correctly.

The URI parsing and manipulation API — including types for scheme, authority, path,
query, and fragment — will land in a subsequent release once the design is finalised.
