kuri parses and canonicalizes untrusted URLs, so security reports are taken seriously.
See [`SECURITY.md`](https://github.com/dexpace/kuri/blob/main/SECURITY.md) for the
supported versions and how to report a vulnerability privately through GitHub's Security
tab — please don't open a public issue for a security problem.
